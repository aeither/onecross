# OneCross

Cross-chain wallet analytics.
- domain names owned
- native token balance
- token balances
- nft balances
- support 8 blockchains

# DEMO

Try it out:

https://prj.vercel.app/

## Setup & Development

Initialize repo in the root:

```
bun install
```

Start development server:

```
bun run dev
```

## Techstack

- nextjs
- bun
- chainbase
